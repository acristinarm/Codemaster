Perguntas sobre HTML

Data do exercício: (22/12/2023)

1 - O que é HTML?

O HTML é uma linguagem de marcação de texto e tem como objetivo formatar a forma como visualizamos o texto, ou seja, o conteúdo.

2 - O que é um elemento HTML? Dê exemplos.

Um elemento HTML é tudo o que fica entre a tag de abertura e a tag de fechamento.
Por exemplo: <p>Eu sou um parágrafo</p>.
Tudo o que está acima é um elemento, ou seja, o conjunto da tag <p> e o conteúdo "Eu sou um parágrafo".

3 - Que ferramentas são necessárias para construir um site HTML?

É necessário um computador e um editor de texto, como o Visual Studio Code.

4 - Quais os elementos base necessários para a criação de uma página HTML?

Os elementos base necessários para construir um site HTML são: <!DOCTYPE html>, <html>, </html>, <head>, </head> e <body>, </body>.

5 - O elemento "br" necessita ser fechado?

Não.

6 - Para que servem os atributos? Dê um exemplo.

Os atributos servem para darmos informação extra a determinada tag e assim ela se comportar de determinada maneira.
Por exemplo: o atríbuto "lang" quando colocado com a tag <html>, no início do código, serve para determinar o idioma em que está escrito o conteúdo.
